Optimal parameters for an OLA filter

This function computes the optimal input block length for an OLA process (see filter_fft()). The optimum is found for \(N_e = 2^k - (M-1)\), \(k\) being found by exhaustive search.

Parameters
MTime-domain filter length (in samples),
[out]CComplexity, in FLOPS by input sample,
[out]NfFFT dimension,
[out]NzNumber of zeros inserted for each FFT block ( \(N_f = N_e + N_z\))
[out]NeOptimal input block length.
