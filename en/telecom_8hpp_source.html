665 // Parameters
666 // x: input signal
667 // σ: square root of noise power
668 //
669 // Description
670 // Compute <latex>$y = x + n$</latex>, with <latex>$n: N(0,\sigma)$</latex>.
671 // If x is a complex signal, or if complex noise is specifically specified,
672 // then noise (with same energy) is also added on the imaginary axis.
673 // So be carefull, with complex noise, the noise power is two times more than for real noise.
674 //

677 // y = awgn(x,0.1);
