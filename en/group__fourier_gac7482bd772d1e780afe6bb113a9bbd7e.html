template<typename derived >
Vector<typename derived::Scalar> dsp::fourier::fftshift ( const Eigen::ArrayBase< derived > &  X)

Spectrum shift

This function shift the input spectrum so as the 0 Hz bin is at the center. If the number of samples is even:

\[ X'_k = X_{k+N/2\ [N]} \]

That is:

\[ X' = \left[X_{N/2}, X_{N/2+1}, \dots, X_{N-1}, X_0, X_1, \dots, X_{N/2-1}\right] \]

And if \(N\) is odd:

\[ X' = \left[X_{N/2+1}, X_{N/2+2}, \dots, X_{N-1}, X_0, X_1, \dots, X_{N/2}\right] \]

Parameters
XInput spectrum, uncentered (typicaly ouput of the fft() function)
Returns
0 Hz centered spectrum.
Example
int n = 10000;
ArrayXf x = bruit_colore(n, 1.0f);
ArrayXf y1 = 10 * fft(x).abs2().log10();
ArrayXf y2 = fftshift(y1);
ArrayXf fr1 = linspace(0, 1.0, n);
ArrayXf fr2 = linspace(-0.5, 0.5, n);
Figures f;
f.subplot().plot(fr1, y1, "b-", "y1 (spectre avant fftshift)");
f.subplot().plot(fr2, y2, "b-", "y2 (spectre apr√®s fftshift)");
