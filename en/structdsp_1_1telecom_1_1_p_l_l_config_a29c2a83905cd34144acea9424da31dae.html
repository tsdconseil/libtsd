Ped dsp::telecom::PLLConfig::ped

Par défaut, le détecteur d'erreur de phase est directement l'argument (y = std::arg(x)). Cette fonction doit retourner un angle en radians.

Exemple 1 (détecteur par défaut) :
config.detecteur_erreur_phase = [](cfloat x)
{
return std::arg(x);
};

Exemple 2 (boucle quadratique) :
Dans cet exemple, le signal est mis au carré avec de la calculer l'argument, ce qui permet de détecter la porteuse résiduelle, même en présence d'une modulation BPSK (ou AM).
config.detecteur_erreur_phase = [](cfloat x)
{
return std::arg(x*x) / 2;
};
