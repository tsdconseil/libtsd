Vector rotation

Parameters
xInput column vector
dInteger number of steps (positive or negative)
Returns
Data vector with rtation of \(d\) steps (modulo the vector dimension):

\[ y_k = x_{k + d\ [N]} \]

\(N\) being the vector length.
Example
ArrayXf x = linspace(0, 5, 6);
ArrayXf y = rotation_vec(x, 2);
ArrayXf yref(6);
yref << 2, 3, 4, 5, 0, 1;
tsd_assert(y.isApprox(yref));
