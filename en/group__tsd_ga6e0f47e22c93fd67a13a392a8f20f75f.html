template<typename T >
auto dsp::diff ( const Vector< T > &  x)

Difference between 2 successive elements of a vector

Returns
A column vector of length \(n-1\):

\[ y_k = x_{k+1} - x_k,\quad k = 0,\dots, n-2 \]

Example
ArrayXf x = randn(10);
ArrayXf y = diff(x);
ArrayXf yref = x.tail(9) - x.head(9);
tsd_assert(y.isApprox(yref));
