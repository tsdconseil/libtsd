Telecommunications

  • Documentation
  • / Telecommunications

#include "dsp/telecom.hpp"

This module contains functions relative to digital and analog transmission, using "software radio".


Sub-groups

Digital modulations
Modulation and demodulation algorithms, simple packet emitter and receiver.

Analog modulations
FM and AM modulation and demodulation.

Shaping filter
Shape binary data as continuous signals.

Plotting
Special plots for telecoms applications.

Simulation
Propagation channels simulation (AWGN, fading, etc.)

Synchronisation sequences
Some codes for synchronisation (maximal length., Barker, etc.)

Error rate measurements
Some facilities for error rate measurements.

AGC and SNR measurements
AGC and SNR measurements.

Channelization
Channelization (frequency multiplexing / demultiplexing) and baseband conversion (image filtering).

Channel equalization
Correction of channel distortions.

Carrier and clock recovery
Recouvrement de l'horloge, de la phase, de la fréquence d'un signal modulé.

Protocols
Quelques protocoles de transmission.

Error correcting codes
Some error correcting codes.
