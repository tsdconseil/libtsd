Structure PacketReceiverConfig

  • Documentation
  • / Telecommunications
  • / Digital modulations
  • / Structure PacketReceiverConfig

Structure de configuration d'un récepteur générique.

Namespace: dsp::telecom

Fields list

format : TrameFormat

Format des trames. 

config_demod : DemodConfig

Configuration du démodulateur. 

BS : int

Dimension des blocs d'entrée (?). 

seuil : float

Seuil de détection pour l'en-tête (entre 0 et 1), voir détecteur_création(). 

SNR_mini : float

Minimal SNR (in dB) detected on the header to decode the frame. 

correl_fft : bool

Si vrai, calcul de la corrélation avec les motifs via des FFT / OLA (sinon filtres RIF). 

callback_corr : fonction< void(const Vecf &c)>

Callback optionnelle appelée avec le signal de corrélation normalisée (peut servir pour de la mise au point). 

debug_actif : bool

Activation ou non des plots de mise au point. 

ncoefs_interpolateur : int

Nombre de coefficient du filtre d'interpolation RIF utilisé avant le démodulateur pour corriger l'horloge.
