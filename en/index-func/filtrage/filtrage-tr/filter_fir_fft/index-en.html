filter_fir_fft

  • Documentation
  • / Linear filtering
  • / Real-time filtering
  • / filter_fir_fft

Efficient implementation of a FIR filter in the frequency domain.

Namespace: dsp::filter

Prototype

sptr<FilterGen<T>> filter_fir_fft(const Vecf &h)

Parameters

hVector with filter coefficients.

Returns

Filter T -> T

Description

This filter implements a FIR filter through the (Ovelap-And-Add) technique. The complexity is thus much lower than the direct implementation if the number of coefficients \(M\) is important (in the order of \(\log M\) operations by sample instead of \(M\)).

Note
Note that this technique introduce a little more delay than the direct, time domain, implementation.
