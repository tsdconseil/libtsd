design_fir_prod

  • Documentation
  • / Linear filtering
  • / Linear filter design
  • / design_fir_prod

Computes the FIR filter equivalent to the serie concatenation of two other FIR filters.

Namespace: dsp::filter

Prototype

Vecf design_fir_prod(const Vecf &h1, const Vecf &h2)

Parameters

h1Coefficients of the first filter ( \(n_1\) taps)
h2Coefficients of the second filter ( \(n_2\) taps)

Returns

Coefficients of the serie concatenation ( \(n_1+n_2-1\) taps)

Description

This function computes the coefficients of a FIR filter that has the same response as the serie cascading of two FIR filters given as parameters: \[ h = h_1 \star h_2 \]

The number of output coefficients is computed as: \(n = n_1 + n_2 - 1\).
