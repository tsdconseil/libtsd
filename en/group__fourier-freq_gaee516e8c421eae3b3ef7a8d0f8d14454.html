Estimation de fréquence

Cette fonction essaye d'estimer la fréquence d'un signal périodique (exponotielle pure).

Note
Si le signal d'entrée est réel (sinusoide pure de fréquence \(f\)), la fonction renvoie une des deux valeurs possibles de fréquence ( \(f\) ou \(-f\)).
Warning
Si le signal d'entrée est composé de plusieurs fréquences distinctes, cette fonction n'est pas adaptée. Utilisez plutôt un algorithme de type sous-espace (voir psd_subspace()).
Parameters
xSignal dont on veut estimer la fréquence
mMéthode d'estimation (voir l'énumération FreqEstimMethode)
Returns
Une fréquence normalisée (entre -0,5 et 0,5)
