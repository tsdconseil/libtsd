Equidistant point interval

Compute \(n\) equidistant points between \(a\) and \(b\).

Parameters
aInitial point (first value)
bFinal point (last value)
nNumber of points
Returns
Interval defined as:

\[ x_k = a + k \cdot \frac{b-a}{n-1},\ k = 0,\dots,n-1 \]

Example 1: vector with the

\[ x_k = a + k \cdot \frac{b-a}{n-1},\ k = 0,\dots,n-1 \]

#1 first integers:
ArrayXf t = linspace(0, n-1, n)
// t = 0, 1, 2, ..., n-1
