◆ filtre_dc()

template<typename T >
sptr<FiltreGen<T> > tsd::filtrage::filtre_dc ( float  fc)

Filtre pour la suppression du DC

Implémente le filtre :

\[ y_k = x_k - x_{k-1} + \alpha y_{k-1} \]

d'après The DC Blocking Filter, J M de Freitas, 2007

Paramètres
fcFréquence de coupure normalisée (entre 0 et 0.5)
Exemple
float fc = 0.01;
int n = 500;
ArrayXf x = 0.2 * sigsin(0.1, n) + 1.0f;
auto filtre = filtre_dc<float>(fc);
ArrayXf y = filtre->step(x);
Figures f;
f.subplot(221).plot(x, "b-", "Signal d'entrée");
f.subplot(222).plot_psd(x, 1, "b-", "Spectre signal d'entrée");
f.subplot(223).plot(y, "b-", "Signal de sortie");
f.subplot(224).plot_psd(y.tail(y.rows()-200), 1, "b-", "Spectre signal de sortie");
