template<typename T >
sptr<Sink<T, int> > tsd::creation_tampon ( int  N,
std::function< void(const Vecteur< T > &)>  callback

Préparation d'un flux de paquets de données de dimension fixes

A partir d'un flux de données entrant, découpé en paquet de dimensions quelconques, ce bloc produit un flux de données sous forme de paquets de dimension fixe.

Paramètres
Ndimension des paquets de sortie
callbackFonction utilisateur qui sera appelée à chaque fois qu'un nouveau paquet de dimension N est disponible.
Renvoie
Un puit de donnée acceptant des vecteurs de type T.
Paramètres du template
TType de données d'entrée / sortie (e.g. float, cfloat, double, ...).
Exemple : préparation des paquets pour une FFT
auto t = creation_tampon<float>(512,
// Définition de la callback
[](const ArrayXf &x)
{
// Ici, on est certain d'avoir 512 échantillons
tsd_assert(x.rows() == 512);
ArrayXcf X = fft(x);
});
// 10000 échantillons à traiter
ArrayXf x(10000);
// x = ...
// Traite les échantillons, par paquets de 512
t->step(x);
tsd::fourier::fft
auto fft(const Eigen::ArrayBase< derived > &x)
Transformée de Fourier Discrète (TFD) rapide.

Notez que 10000 n'étant pas un multiple de 512, les derniers échantillons ne sont traités (ils le seront si la méthode t->step()) est appelée à nouveau avec d'autres échantillons).
