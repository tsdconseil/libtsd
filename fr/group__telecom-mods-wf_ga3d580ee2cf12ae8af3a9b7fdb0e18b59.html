Modulation de phase

Création d'une forme d'onde M-PSK. Le résultat peut être utilisé pour créer un modulateur (modulateur()) ou un démodulateur (demodulateur()).

Paramètres
MNombre de bits / symboles.
filtreFiltre de mise en forme (par défaut : NRZ)
Renvoie
Pointeur vers une forme d'onde abstraite (FormeOnde).
Avertissement
\(M\) doit être une puissance de 2.
Exemple 1 : tracé de quelques constellations PSK
Figures f;
for(auto M: {2, 4, 8, 16})
{
auto w = waveform_psk(M);
auto constel = w->constellation();
f.subplot().plot_iq(constel, "o", w->desc());
}
