sousech

  • Documentation
  • / Fonctions générales
  • / sousech

Sous-échantillonnage d'un vecteur colonne.

Espace de nom : tsd

Prototype

auto sousech(const Vecteur<T> &x, entier R)

Description

A partir d'un vecteur \((x_k), k=0\dots n-1\), renvoie un vecteur sous-échantillonné d'un facteur \(R\) : \[ y_k = x_{kR},\quad k =0 \dots \left\lfloor N/R\right\rfloor -1 \]

Exemple :

  soit x = linspace(1, 20, 20);
  soit y = sousech(x, 2);

  Figure f;
  f.plot(x, x, "b-", "x");
  f.plot(y, y, "or", "y");
