vconcat (1)

  • Documentation
  • / Fonctions générales
  • / vconcat (1)

Concaténation verticale de deux vecteurs.


Prototype

auto vconcat(const Vecteur<T> &a, const Vecteur<T> &b)

Paramètres

aPremier vecteur
bDeuxième vecteur

Retourne

Concaténation verticale

Description

Cette fonction réalise l'assemblage de deux vecteurs colonnes : \[ c = \left( \begin{array}{c} a_0\\ a_1\\ \vdots\\ b_{n-1}\\ b_0\\ b_1\\ \vdots\\ b_{m-1}\\ \end{array} \right) \]

(en supposant que a et b aient respectivement \(n\) et \(m\) éléments). L'opérateur | a été redéfini pour réaliser la même opération.

Exemple

soit a = Vecf::ones(1),
     b = Vecf::zeros(5),
     c = vconcat(a, b); // (ou c = a | b)
