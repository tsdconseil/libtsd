design_rif_gaussien

  • Documentation
  • / Filtrage linéaire
  • / Design de filtres
  • / design_rif_gaussien

Coefficients d'une approximation RIF d'un filtre gaussien (non fenêtré).

Espace de nom : tsd::filtrage

Prototype

Vecf design_rif_gaussien(entier n, float σ)

Paramètres

nNombre de coefficients du filtre.
σEcart-type (en nombre d'échantillons).

Description

Exemple

  soit h1 = design_rif_gaussien(21, 0.5),
       h2 = design_rif_gaussien(21, 2);

  Figures f;
  f.subplot().plot(h1, "|ob", "σ = 0,5");
  f.subplot().plot(h2, "|og", "σ = 2,0");
