Affichage à l'écran dans une fenêtre à part.


Prototype

void afficher(cstring titre="", const Dim &dim={-1, -1}) const

Paramètres

nom_fenetreTitre de la fenêtre graphique.

Description

Avertissement
Cette fonction n'aura aucun effet si la dépendance optionnelle GTKMM n'est pas présente au moment de la compilation de la librairie.

Figure f;
f.plot(x);
f.afficher();
