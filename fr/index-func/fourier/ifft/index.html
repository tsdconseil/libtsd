Transformée de Fourier Discrète Inverse.

Espace de nom : tsd::fourier

Prototype

auto ifft(const Vecteur<T> &X)

Description

Cette fonction calcule la TFD inverse normalisée d'un signal réel ou complexe : \[ X_n = \frac{1}{\sqrt{N}}\cdot\sum_{k=0}^{N-1} x_k \cdot e^{\frac{+2\pi\mathbf{i}kn}{N}} \]

Exemple

soit x1 = randcn(100),
     X  = fft(x1),
     x2 = ifft(X);

assert((x2-x1).norm2() < 1e-7);
