◆ modele_constante()

sptr< SSMLineaire > tsd::kalman::modele_constante ( float  R = 0.1)

Valeur constante

Cette fonction renvoie le système en représentation d'état le plus simple : une valeur constante, observée à travers un canal bruité, c'est-à-dire :

\[ x_n = cst,\quad y_n = x_n+v_n \]

avec \(v_n: \mathcal{N}(0,R)\)

Paramètres
RVariance du bruit d'observation
Renvoie
Modèle linéaire
Exemple :
auto sys = modele_constante();
// Valeur initiale de l'état = 1
VectorXf x0 = 1 * VectorXf::Ones(1);
// Simulation : 50 itérations
auto [x,y] = sys->steps(50, x0);
Figure f;
f.plot(x.row(0), "b-o", "Vrais états");
f.plot(y.row(0), "m-o", "Observations");
