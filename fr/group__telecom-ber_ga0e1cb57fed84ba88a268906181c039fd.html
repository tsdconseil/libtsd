Comparaison de chaines binaires

Try to find the best correlation between the 2 bit vectors and count the number of errors (ignoring the 2 first bits and 2 last bits).

Paramètres
b0Premiere chaine (vecteur de 0 et 1)
b1Deuxième chaine (idem)
Exemple
b1 = [0 1 0 0 0 1];
b2 = [1 0 0 0 1];
auto res = cmp_bits(b1,b2);
// res.ber is the bit error rate
