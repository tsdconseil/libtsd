)

Conversion train binaire \(\to\) index

Cette fonction convertit un vecteur binaire (valeurs : 0 ou 1), en un vecteur de symboles, avec \(k\) bits / symboles, suivant l'encodage binaire standard :

\[ y_i = \sum_{j=0}^{k-1} x_{ki+j} 2^j,\quad i = 0,\dots,(n+k-1)/k \]

Paramètres
xSignal d'entrée (valeurs : 0 ou 1)
kNombre de bits par symbole
Renvoie
Vecteurs de \((n+k-1)/k\) échantillons.
Note
Si le nombre d'échantillons \(n\) du signal d'entrée n'est pas un multiple de \(k\), des zéros sont insérés à la fin de manière à produire le dernier échantillon.
