Complétion de vecteurs avec des zéros

Un des deux vecteurs est complétés avec des zéros de manière à ce que les deux vecteurs résultants aient la même longueur.

Paramètres
xPremier vecteur
yDeuxième vecteur
p2Si vrai, arrondi la dimension des vecteurs de sortie à la prochaine puissance de 2.
Renvoie
Un tuple de deux vecteurs de mêmes dimensions, et commençant respectivement par x et y (des zéros sont ajoutés à la fin).
Exemple
auto x = linspace(0,4,5);
// x = {0, 1, 2, 3, 4};
auto y = linspace(0,3,4);
// y = {0, 1, 2, 3};
auto [x2,y2] = pad_zeros(x,y);
// x2 = {0, 1, 2, 3, 4};
// y2 = {0, 1, 3, 4, 0};
