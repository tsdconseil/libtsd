Concaténation verticale

Cette fonction réalise l'assemblage de deux vecteurs colonnes :

\[ c = \left( \begin{array}{c} a_0\\ a_1\\ \vdots\\ b_{n-1}\\ b_0\\ b_1\\ \vdots\\ b_{m-1}\\ \end{array} \right) \]

(en supposant que a et b aient respectivement \(n\) et \(m\) éléments). L'opérateur | a été redéfini pour réaliser la même opération.

Paramètres
aPremier vecteur
bDeuxième vecteur
Renvoie
Concaténation verticale
Example
ArrayXf a = ArrayXf::Ones(1);
ArrayXf b = ArrayXf::Zero(5);
ArrayXf c = vconcat(a, b); // (ou ArrayXf c = a | b)
